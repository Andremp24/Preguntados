   


El desalloro del reto se basa en hacer varias clases y cada una de ellas una tarea especifica.

Niveles = En cual esta cada rondaa y como se va a ir desarollando el juego y es paquete donde corre la gran mayoria del cogido del juego. 

Preguntas = en esta clase se copio mas que todo todas las 25 preguntas creando un matriz para poner recorreras y llamarlas en niveles. 

Registro = ES donde se guarda el historial del juego el cual nos llega el nombre del ususario y el score que saca por cada pregunta.

Usuarios = recibe el nombre y se hace el score de cada jugador para enviarlos como parametros  creando objetos de esta clase en otras claves para su invocacion.

Cuenta con una clase principal donde solo se van a invocar o crear objetos de las clases anteriores para imprimirlo.


Se crea un archivo  llamado Marcador.txt que va a guardar el nombre que se registra y el score logrado por el usuario.
